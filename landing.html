✨ What it does

Built around real academic content

Every feature maps directly to the syllabus — Least Squares Estimation, model evaluation, hypothesis testing, and visualization — implemented in pure vanilla JavaScript.

📂
Data Loading
Load built-in datasets (House Price, Car Price), upload a custom CSV, or enter your own rows by hand — then preview the data in a clean table.
🧠
LSE Regression
Simple & Multiple Linear Regression via Least Squares Estimation — explicit formulas for the simple case and matrix form β = (XᵀX)⁻¹XᵀY for the multiple case. No external math libraries.
📈
3 Live Charts
Scatter plot with regression line, Actual vs. Predicted comparison, and Residuals distribution — all rendered live with Chart.js.
📋
Full Evaluation
R², MSE, RMSE, Standard Error of estimate, the final regression equation, ANOVA table, t-Test for coefficients, and 95% Confidence Intervals.
✏️
Manual Data Entry
Type column names and rows directly into the app — useful for small classroom examples without preparing a CSV file.
💡
Intelligent Recommendations
Strength of relationships, most influential features, model suitability, and data quality suggestions — generated automatically from the trained model.
